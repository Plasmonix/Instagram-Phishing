## 🛠 Installation & Usage

> **☕ Downloading**
```
git clone https://github.com/Plasmonix/Instagram-Phishing
cd Instagram-Phishing
```

> **🧾 Requirements**
```
pip install flask
```

> **🖥️ Start**
```
python app.py
Open web browser and navigate to localhost:5000
```
